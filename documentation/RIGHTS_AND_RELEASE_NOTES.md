# Rights And Release Notes

The package payload is a legacy author/project-created Excel/VBA model. No raw person-level data, GPS traces, third-party source data, correspondence, drafts, or presentation files are included.

The original workbooks should be released with clear provenance notes and an explicit license decision for the author-created workbook model/code. The article PDF in `paper/` is included only as a local audit reference and should not be publicly uploaded unless publisher rights review clears it.

The shared `Qp2`, `Qpi2`, and `ServiceTime` workbooks should be cited from `_shared_sources/probe-atis-spreadsheet-model` rather than recopied into this repository.
